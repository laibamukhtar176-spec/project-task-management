# Project & Task Management System (V1.0)
Developed for Allied Software Engineers.

## Live Deployment Link
👉[https://laibamukhtar176-spec.github.io/project-task-management/]

## Key Features Implemented
- **Login Panel:** Complete authentication verification layout.
- **Dashboard Tracking:** Live tracking of completion, pending status, and active items.
- **Strict Logic Rules:** System flags overdue processes and bars faulty date entry arrays.
- **Bonus Addition:** Custom JavaScript interactive status progress metric tracking.

## Git Workflow & Branching Strategy Used
- `main` - Stable production code.
- `development` - Shared integration space for all feature branches.
- `feature-login`, `feature-dashboard`, `feature-task-module` - Isolated feature workspaces.
- `release-v1.0` - Pre-production compilation track.
- `hotfix-validation` - Emergency debugging channel for validation rules.
